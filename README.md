# MAPGR: Multi-Agent Prompt-Guided Residual Diffusion for Ancient Mural Restoration

This is the source code for **'MAPGR: Multi-Agent Prompt-Guided Residual Diffusion for Ancient Mural Restoration'**. We provide the code, the dataset, and the pretrained models. For convienence of introduction, we name the proposed network as MAPGR.

## 🛠️ Installation

### 1. Environment Setup
* Python 3.10
* PyTorch 2.1.2
* CUDA 12.1
* We run the codes on NVIDIA A100-PCIE-40GB

### 2. Datasets
The datasets analyzed during the current study are publicly available to support the transparency and reproducibility of our findings. The [DUNHUANG](https://www.kaggle.com/datasets/xuhangc/dunhuang-grottoes-painting-dataset-and-benchmark) dataset, which provides domain-specific stylistic distributions of ancient grotto paintings. The [DhMurals](https://1drv.ms/u/s!AittnGm6vRKLzXorf1nkiDPRQB4D?e=Avv27i) dataset, comprising a heterogeneous collection of mural images for structural evaluation.
